### Node Version Manager

#### Install NVM
```
# Download and run script
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# source zsh runtime
source ~/.zshrc
```

#### List Avaiable Node.js LTS
```
nvm ls-remote --lts
```

#### Install Node.js v20.15.0 LTS
```
nvm install 20.15.0
```
