#### Adjusting pane size
- resize up 5 rows -- `Ctrl`+`B` -> `Shift`+`:` -> resize-p -U 5 
- resize down 5 rows -- `Ctrl`+`B` -> `Shift`+`:` -> resize-p -D 5
- resize left 5 rows -- `Ctrl`+`B` -> `Shift`+`:` -> resize-p -L 5
- resize right 5 rows -- `Ctrl`+`B` -> `Shift`+`:` -> resize-p -R 5
